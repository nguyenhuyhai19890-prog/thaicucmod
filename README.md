# Thai Cuc Mod - Yin Yang Legendary Sword

Minecraft 1.21.1 Fabric mod featuring Thai Cuc Kiem (Yin Yang Sword) with legendary abilities.

## Features

### Thai Cuc Kiem Sword
- Black and white yin-yang design
- Netherite-based weapon
- Two powerful skills:

**Skill 1: Meditation Slash (Right Click)**
- Enter meditation pose
- 1 second wind-up
- AOE slash attack (15 block range)
- Damage: 15 HP
- Cooldown: 15 seconds

**Skill 2: Thai Cuc Territory (Z Key)**
- Create massive yin-yang domain
- 20 block radius effect area
- Continuous enemy damage
- Player buffs: Speed + Resistance
- Duration: 15 seconds
- Cooldown: 60 seconds

## Installation

1. Java 21+
2. Minecraft 1.21.1
3. Fabric Loader + Fabric API

## Build

```bash
./gradlew build
```

Mod jar: `build/libs/thaicucmod-1.0.0.jar`

## Usage

```
/give @s thaicucmod:thai_cuc_sword
```

- Right Click: Meditation Slash
- Z Key: Thai Cuc Territory
